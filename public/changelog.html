4.0.0 Jun 19, 2018

  • Updated Esri JS Api from 3.x to 4.x
  • Implemented Babel (#195).
  • Upgraded from MagicZoom to Sherlock.
  • Implemented eslint-agrc-config.
  • Upgraded from agrc.widgets to dart-board (FindAddress) and map-tools (MapView & extentRouter).
  • Added link to Google Street View to popup.
  • Implemented Esri Print Widget.
  • Implemented htmlhint.
  • Ditch yarn for npm.
  • Removed newer grunt task from eslint.
  • Upgraded eslint from 3.x to 4.x.
  • Upgrade grunt-contrib-jasmine thus kicking phantomjs to the curb.
  • Switch from bower to npm for front end dependency management.
  • Switched from eslint preset latest to env.
  • Remove proj4 dependency in favor of esri/geometry/projection.
  • Removed EmbeddedMapLoader.js (not used for years).
  • Make slightly more mobile friendly by hiding the sidebar (on load) and AGRC logo on smaller width devices.
  • Randomly choose from the capitols of the four most populous counties as the default extent on load.
  • Persist map extent between sessions (via local storage)
  • Upgrade all code to use modern ES6 JavaScript sytax.
  • Implement grunt-cache-breaker to help make sure that users always have the latest version deployed to the server (#148).
  • Add COUNTY as a context field for the Points of Interest search (#150).
  • Implement map-tools/extentRouter which adds URL parameters for determining the current map extent (#202).
